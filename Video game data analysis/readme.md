# Секреты "Темнолесья" 

[HTML] [ipynb]
## Описание проекта

Команда игры «Секреты Темнолесья» хочет подготовить статью о развитии индустрии игр в начале XXI века, с от 2000 до 2013 года включительно. Данныме собрали из открытых источников. В них есть информация о продажах игр, сделанных в разных жанрах и выпущенных на разных платформах, а также пользовательские и экспертные оценки игр. Необходимо познакомиться с данными, проверить их корректность и провести предобработку, получив необходимый срез данных.

## Навыки и инструменты
+ **pyton**
+ **pandas**

## Общий вывод
Данные выгружены из файла .csv и проанализированы. 
Датафрейм предобработан, наименования столбцов приведены к единому стилю snake case, столбцы приведены к нужным типам, пропуски в данных заменены.
Удалены явные дубликаты. 
В соответствии с заданием получен срез данных с годом релиза игры от 2000 до 2013 включительно. Игры разделены по категориям. Получен список ТОП-7 игровых платформ.

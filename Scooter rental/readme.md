# "Анализ данных и проверка гипотез в бизнесе проката самокатов"
[HTML](https://github.com/nina-moise/practicum_projects/blob/main/Startup%20analysis/Startup%20analysis.html) [ipynb](https://github.com/nina-moise/practicum_projects/blob/main/Scooter%20rental/Scooter%20rental.ipynb)
## Описание проекта
Крупный сервис проката самокатов GoFast предоставил три датасета с данными о пользователях сервиса, их поездках за один год, а также стоимости услуг в подписках. Необходимо проанализировать демографию пользователей и особенности использования самокатов, а также определить возможную выгоду от распространения платной подписки на самокаты.

## Навыки и инструменты
+ **Python**
+ **Pandas**
+ **Matplotlib**
+ **Scipy**
 
## Общий вывод
В проекте проделана следующая работа: данные загружены в датафреймы, предобработаны, удалены дубликаты.

Проведен разведочный анализ данных, который ответил на поставленные бизнесом вопросы:
о количестве и возрасте пользователей, месте проживания, количестве пользователей с платной подпиской, средней долительности поездки для разных типов пользователей, выручки на каждого пользователя.

Дальнейший анализ и проверка гипотез помогли выявить ключевые характеристики распределения данных и определить критические значения для принятия эффективных бизнес-решений.

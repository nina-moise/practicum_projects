## Цель исследования 
Провести расчёт и анализ ключевых метрик: LTV, CAC, ROI, ARPPU и Retention Rate — и составить заключение о текущем состоянии юнит-экономики приложения.

+ **Pandas**

+ **Unit Economics**
+ **Metric Tree Decomposition**
+ **Cohort Analysis**
  
## Результаты исследования
Загружен набор данных из 4 таблиц. В динамике по месяцам, когортам и каналам привлечения рассчитаны и проанализированы ключевые бизнес-метрики юнит-экономики, оценена окупаемость бизнеса.

**Ключевой вывод:** Юнит-экономика проекта находится в состоянии неустойчивого равновесия. Рост доходности с одного игрока полностью поглощается растущей стоимостью привлечения.
-   Эффективность маркетинга (CAC): Наблюдается негативный тренд — стоимость привлечения пользователя растет. Это основной риск для масштабирования игры.
-   Качество продукта (LTV & ARPPU): Положительная динамика. Продукт успешно монетизирует аудиторию (рост среднего чека), а лояльность (Retention) остается стабильной.
-   Окупаемость (ROI & Payback Period): Метрики стагнируют. Текущий рост LTV едва компенсирует инфляцию CAC, из-за чего общая прибыльность инвестиций не растет.

## Рекомендации:

-  Работа с воронкой: использовать дерево метрик для поиска узких мест в конверсии на этапе оплаты, чтобы ускорить окупаемость (Payback Period).

# Мобильная игра "Mines & Moles"
[HTML](https://github.com/nina-moise/practicum_projects/blob/main/Mines%20%26%20Moles/Mines%20%26%20Moles.html) [ipynb](https://github.com/nina-moise/practicum_projects/blob/main/Mines%20%26%20Moles/Mines%20%26%20Moles.ipynb)

## Цель исследования 
Поиск и факторный анализ причин падения выручки: от расчёта продуктовых метрик до визуализации и интерпретации результатов на основе набора данных поведения пользователей в приложении мобильной игры. 

+ анализ дерева метрик,
+ формулирование гипотез причин падения выручки,
+ выбор бизнес-метрик для проверки гимпотез,
+ расчет и визуализация бизнес-метрик в динамике, 
+ анализ метрик и выводы.

## Результаты исследования
Загружен набор данных из 5 таблиц. В динамике по месяцам рассчитаны и проанализированы ключевые бизнес-метрики. Найдена основная причина падения выручки, даны рекомендации.

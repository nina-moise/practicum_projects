# Мобильная игра "Mines & Moles"
[HTML]() [ipynb]()

## Цель исследования 
Поиск и факторный анализ причин падения выручки: от расчёта продуктовых метрик до визуализации и интерпретации результатов на основе набора данных поведения пользователей в приложении мобильной игры. 

+ анализ дерева метрик,
+ формулирование гипотез причин падения выручки,
+ выбор бизнес-метрик для проверки гимпотез,
+ расчет и визуализация бизнес-метрик в динамике, 
+ анализ и выводы.

Набор данных из 5 таблиц загружен. В динамике по месяцам рассчитаны и проанализированы ключевые бизнес-метрики. Найдена основная причина падения выручки, даны рекомендации.

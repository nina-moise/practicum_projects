# Решение Ad hoc задач в сервисе Яндекс Книги

**Работа с данными бизнеса в ClickHouse**

[HTML](https://github.com/nina-moise/practicum_projects/blob/main/Yandex%20Books%20Ad%20hoc%20ClickHouse/Yandex%20Books%20Ad%20hoc%20ClickHouse.html) [ipynb](https://github.com/nina-moise/practicum_projects/blob/main/Yandex%20Books%20Ad%20hoc%20ClickHouse/Yandex%20Books%20Ad%20hoc%20ClickHouse.ipynb)
## Цель проекта 
Решение ad hoc задач по аналитике, посвященных основным метрикам и паттернам использования сервиса. Анализируются мобильные платформы 'Букмейт iOS' и 'Букмейт Android', в некоторых случаях 'Букмейт Web'.
Основной инструмент решения задач — ClickHouse. Работаем с двумя таблицами в DBeaver.

## Навыки и инструменты
+ **SQL**
+ **ClickHouse**
+ **DBeaver**
  
## Результаты 
Решены 11 Ad hoc задач, поставленных бизнесом. Решение оформлено в тетрадке Jupyter Notebook.

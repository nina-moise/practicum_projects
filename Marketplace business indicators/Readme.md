# Бизнес-показатели маркетплейса
[HTML](https://github.com/nina-moise/Portfolio/blob/main/Marketplace%20business%20indicators/Marketplace%20business%20indicators.html) [ipynb](https://github.com/nina-moise/Portfolio/blob/main/Marketplace%20business%20indicators/Marketplace%20business%20indicators.ipynb)
## Цель исследования 
На основе набора данных построение целостной картины бизнеса:
+ обзор ключевых метрик,
+ оценка монетизации (как продукт зарабатывает деньги),
+ оценка юнит-экономики (насколько это прибыльно в расчете на единицу клиента или товара),
+ поиск инсайтов и точек роста для бизнеса.

## Навыки и инструменты
+ **pyton**
+ **pandas**
+ **seaborn**
+ **matplotlib**
+ **юнит-экономика**
+ **когортный анализ**
  
## Результаты исследования
Набор данных из 4 таблиц загружен, предобработан. Построено дерево метрик. В динамике по месяцам рассчитаны ключевые бизнес-метрики: GMV, CR, RR, AOV, ARPU, ARPPU. На основе расчетов сделан вывод о состоянии бизнеса.

# Анализ результатов A/B-тестирования нового интерфейса интернет-магазина
[HTML](https://github.com/nina-moise/Portfolio/blob/main/BitMotion%20Kit%20A-B-Test/BitMotion%20Kit.html) [ipynb](https://github.com/nina-moise/Portfolio/blob/main/BitMotion%20Kit%20A-B-Test/BitMotion%20Kit.ipynb)
## Описание проекта
Цель исследования: на основе набора данных и технического задания проверить результаты проведенного А/В теста, подтвердить или опровергнуть гипотезу, что новый интерфейс сайта интернет-магазина **BitMotion Kit** в течение 7 дней после регистрации в системе повысил конверсию зарегистрированных пользователей в покупку не меньше, чем на 3%.

## Навыки и инструменты
+ **pyton**
+ **pandas**
+ **matplotlib**
+ **matplotlib-venn**
+ **калькулятор Эвана Миллера**
+ **statsmodels.stats.proportion.proportions_ztest**

## Общий вывод
Набор данных был загружен и проверен на корректность проведения А/В теста. Сформулированы основная и альтернативная гипотезы. Расчитаны параметры теста. Проведен статистический тест и получены статистически значимые результаты. На основе статистического теста и технического задания сделаны выводы, даны рекомендации.

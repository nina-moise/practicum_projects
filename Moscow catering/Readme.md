
# Проект "Заведения общественного питания г.Москвы"
[HTML](https://github.com/nina-moise/practicum_projects/blob/main/Moscow%20catering/Moscow%20catering.html) [ipynb](https://github.com/nina-moise/practicum_projects/blob/main/Moscow%20catering/Moscow%20catering.ipynb)
## Описание проекта
Исследование и анализ рынка общественного питания города Москвы с целью выбора подходящего места для нового заведения - кафе, ресторана или бара. Выбор для нового заведения меню и ценовую политику.

## Навыки и инструменты
+ **Python**
+ **Pandas**
+ **Matplotlib**
+ **Seaborn**
+ **Phik_matrix**
+ **Numpy**

## Общий вывод
Данные выгружены и предобработаны. Проведен исследовательский анализ данных. 
Даны ответы на следующие вопросы:

* Какие категории заведений общепита встречаются;
* Распределение количества заведений по административным районам Москвы;
* Соотношение сетевых и несетевых заведений;
* Типичное количество посадочных мест в заведениях разного типа;
* Каков пользовательский рейтинг заведений?
* С чем коррелирует рейтинг заведения?
* Топ-15 популярных сетей в Москве;
* Вариация среднего чека заведения в зависимости от района Москвы.

Построены визуализции. Даны рекомендации по району, среднему чеку и ценовой политике нового заведения.

# Проект "Исследование рынка стартапов"
[HTML](https://github.com/nina-moise/practicum_projects/blob/main/Startup%20analysis/Startup%20analysis.html) [ipynb](https://github.com/nina-moise/practicum_projects/blob/main/Startup%20analysis/Startup%20analysis.ipynb)
## Описание проекта
Представленный датасет с информацией о стартапах, компаниях и людях нужно:

1. Исследовать на корректность и полноту представленные исторические данные.
2. Подготовить датасет для заказчика, чтобы в дальнейшем на его основе можно было проводить анализ.
3. Ответить на вопросы заказчика для его дальнейшего входа на инвестиционный рынок.

## Общий вывод
Данные выгружены и предобработаны. Проведен исследовательский анализ данных на полноту и корректность. 
Даны ответы на следующие вопросы заказчика:

* Типичный размер средств, выделяемый в рамках одного раунда;
* Общее количество раундов финансирования за год;
* Зависит ли наличие сведений об образовании работников от размера компании;
* Топ-4 компаний, купленных за 0 или 1 доллар;
* Получены 7 категорий компаний с типично высокими ценами;
* Получен список компаний с высоким разбросом цены;
* Среднее значение количества раундов для каждого возможного статуса стартапа.

Построены визуализции. Даны рекомендации по покупке стартапа с типичной суммой финансирования, определено типичное количество раундов финансирования.
